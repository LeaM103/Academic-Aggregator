This web application runs using a multi-port architecture. The frontend runs on port 3000 and handles the user interface. All search requests are sent to a backend API server running on port 3001, which communicates with external APIs (such as the ArXiv API) and returns processed results. A proxy/load balancer runs on port 8080 and routes incoming traffic to the correct service.
